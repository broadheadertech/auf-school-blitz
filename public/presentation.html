Security

Row-Level Security

Every table has RLS. Students see only their data. Faculty see their sections. Admin sees everything.

Route Guards

Router-level role guards. Admin pages inaccessible to students and faculty.

Database

25+ Postgres Tables

Normalized schema. Foreign keys, indexes, constraints, enums, JSONB fields.

Realtime Channels

Supabase postgres_changes for live messaging and auto-refresh on mutations.

Design

Design System

Consistent, accessible, and responsive across all devices.

🎨

Navy & Gold Theme

Primary: #0077B6 (deep navy). Accent: #C8A415 (gold). Refined academic aesthetic.

🌙

Dark & Light Mode

Full theme support via CSS variables. Persistent user preference in localStorage.

📱

Responsive Design

Desktop sidebar, tablet adaptive, mobile bottom nav. Touch-friendly (44px targets).

🔤

Inter + Space Grotesk

Inter for body text. Space Grotesk for display headings. Clean, modern, legible.

🧩

Component Library

Button, Card, Badge, Input, Modal. Consistent variants, sizes, and states.

♿

Accessibility

Semantic HTML, ARIA labels, focus indicators, reduced motion support, scope headers.
